Principais formatações

Vou utilizar somente tags semânticas

Negrito / Destaque

Nesta frase, tenho um termo em destaque usando a tag <STRONG>(semântica)

Itálico / Ênfase

Nesta frase, temos um termo em ênfase usando a tag <EM>

Texto marcado

Podemos criar também um texto marcado usando a tag <MARK>

Texto grande e pequeno

Podemos criar também um texto grande e um texto pequeno nesse parágrafo

Texo deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Texto sobrescrito

Para inserir coisas do tipo como x20+3

Texto subscrito

Para inserir coisas do tipo como H2O
